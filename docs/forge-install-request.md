# Forge アプリ「sales-request-assistant」プロダクション インストール依頼

## 依頼内容

Forgeアプリ「sales-request-assistant」をプロダクション環境にデプロイ済みです。
管理者コンソールでのインストール承認をお願いします。

## アプリ概要

JPREQプロジェクトに起票された依頼チケットの情報不足をAIが自動チェックし、
不足項目の質問と改善案をコメント投稿するアプリです。

## 承認が必要な権限

| 権限 | 用途 |
|------|------|
| read:jira-work | チケット内容の読み取り |
| write:jira-work | レビュー結果のコメント投稿 |
| 外部通信: https://api.openai.com | AI分析のため |

## 影響範囲（重要）

- **Forgeの仕様上、インストールはサイト全体（epark-tech.atlassian.net）に対して行われます**
- ただし、**アプリ内部のロジックで JPREQ プロジェクトのみに動作を制限**しています
- JPREQ以外のプロジェクトのチケットには一切反応しません（レビュースキップ処理実装済み）
- 将来的に対象プロジェクトを追加する場合は環境変数の変更のみで対応可能です

## 動作内容

1. **自動レビュー（Trigger）**: JPREQにチケットが起票されると自動でレビューコメントを投稿
2. **手動レビュー（Issue Panel）**: チケット画面のパネルから任意のタイミングで再レビュー可能

## 検証状況

- 開発環境（development）で動作確認済み
- 自動テスト全25件パス
- 対象プロジェクト制御の動作確認済み（JPREQ以外はスキップ）

## 管理者側の作業

管理者コンソール（アプリ管理画面）で本アプリのインストールリクエストを
承認していただくだけで完了です。CLI操作等は不要です。
