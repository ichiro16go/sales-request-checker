# Privacy Policy - sales-request-assistant

Last updated: 2026-06-18

## Overview

sales-request-assistant は EPARKグループ内部で使用する Jira Forge アプリです。

## 収集するデータ

- Jira チケットの内容（summary, description, reporter, priority, duedate）
- 上記データは OpenAI API に送信され、レビュー結果生成に使用されます

## データの保存

- 本アプリはデータを永続的に保存しません
- OpenAI API への送信は処理時のみで、保存・学習には使用されません（API利用規約に準拠）

## 利用範囲

- 本アプリは epark-tech.atlassian.net 内部でのみ使用されます
- 外部への公開・配布は行いません

## 連絡先

本アプリに関するお問い合わせは Jira プロジェクト EPGPRD にてご連絡ください。
